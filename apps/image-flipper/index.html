About the app

It's important for Web Developers to understand the basics of manipulating images since rich web applications rely on images to add value to the user interface and user experience (UI/UX).

This Image Flipper explores one aspect of image manipulation - image rotation. This app displays a square pane containing a single image presented in a 2x2 matrix. Using a set of up, down, left, and right arrows adjacent to each of the images the user may flip them vertically or horizontally.

You can change the default image by entering the URL of a different image in an input field and clicking display. Note: this app wasn't designed for mobile first, so it's better suited for desktops.
